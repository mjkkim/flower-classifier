## 프로젝트 개요
- **목적** : 간단한 이미지 분류 모델 제작 및 소켓 통신을 이용한 바이너리 데이터(이미지) 전송 구현
- **개발 기간** : 2023.11 ~ 2023.12

## 기술 스택
**Backend & AI**
- Python 3.12
- **Library** : Tensorflow 2.17.0, NumPy, Pandas
- **Network** : Socket (TCP/IP)
  
**Frontend**
- **GUI** : Tkinter
- **Image Processing** : Pillow

## 주요 기능
- 수신된 이미지를 모델 규격(299x299)에 맞게 리사이징 및 Normalization 수행
- 접속 중인 클라이언트 목록 및 분류 이력을 실시간 로그로 출력
- 소켓을 통해 이미지 데이터를 서버로 전송하고 결과를 수신

## 트러블슈팅
**1) TCP 데이터 파편화 해결**
- **문제** : 아주 드물게, 큰 이미지 데이터를 송신할 때 한 번에 모두 수신되지 않아 데이터가 깨지는 현상 발생
- **해결** : 데이터 전송 전, 8바이트의 고정 크기의 헤더에 파일 크기를 먼저 전송하고, 수신측에서는 해당 크기만큼 반복해서 수신하도록 설계하여 무결성을 보장함

**2) GUI 응답성 유지**
- **문제** : 이미지 분석 및 네트워크 통신 중에 Tkinter GUI가 Freezing 상태가 됨.
- **해결** : `threading`모듈을 활용하여 네트워크 통신 로직을 별도 스레드에서 실행하도록 하여 해결함.
